Use this guide to connect Auth0 to Lovable using OIDC.

## Prerequisites

- Auth0 tenant admin access
- Lovable workspace Owner or Admin

## Configure in Auth0

<Steps>
  <Step title="Create an application" stepNumber="1">
    Auth0 Dashboard → Applications → Applications → Create Application → Regular Web Application.
  </Step>
  <Step title="Allowed Callback URLs" stepNumber="2">
    Add:
    
    - https://auth.lovable.dev/\_\_/auth/handler
  </Step>
  <Step title="Allowed Logout URLs (optional)" stepNumber="3">
    Add your Lovable app logout URL if desired.
  </Step>
  <Step title="API and scopes" stepNumber="4">
    Scopes required:
    
    - `openid`
    - `email`
  </Step>
  <Step title="Copy values to Lovable" stepNumber="5">
    From Settings:
    
    - Domain
    - Client ID
    - Client Secret
    
    Enter these in Lovable → Settings → Workspace → Identity.
  </Step>
</Steps>

## Reference

- Redirect URI: https://auth.lovable.dev/\_\_/auth/handler
- Required scopes: `openid`, `email`

### Troubleshooting

<AccordionGroup>
  <Accordion title="Callback URL mismatch">
    Ensure the callback URL exactly matches https://auth.lovable.dev/\_\_/auth/handler.
  </Accordion>
  <Accordion title="Email not returned">
    Ensure the user has a verified email and the `email` scope is requested.
  </Accordion>
</AccordionGroup>
