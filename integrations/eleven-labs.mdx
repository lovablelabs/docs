## **What ElevenLabs enables**

The ElevenLabs connector adds **realistic text-to-speech, voice cloning, and audio generation** to Lovable apps. Once connected, your app can:

- Read content aloud using high-quality synthesized voices
- Create unique characters or branded narrators
- Generate podcasts, audio summaries, or sound effects
- Power real-time conversational experiences

ElevenLabs turns prototypes into polished, voice-first products.

## **Common use cases**

- **Daily briefing apps** that read out metrics or summaries
- **Voice-enabled assistants** (JARVIS-style interaction)
- **Podcast or audio-story generation**
- **Children’s story creators** with character voices + SFX
- **Accessibility features** (hands-free or screen-free modes)
- **Interview practice coaches** with real-time spoken feedback

## **Example app workflows**

- Add a **“Play” button** to any view
- Let users select narrator style
- Convert any generated text into MP3 audio
- Create conversational experiences by pairing with OpenAI
- Build voice changers or cloning tools

## **Configuration**

1. Go to **Workspace Settings → Integrations**
2. Connect **ElevenLabs**
3. Input your API key
4. Connector becomes available to all projects

You can also configure this through chat when the agent attempts to generate audio.
